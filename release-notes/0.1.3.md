# 0.1.3

- 修复 `0.1.2` 拉取到更新清单后因 manifest 结构兼容问题导致的验签失败，旧版现可正常识别 `0.1.3` 更新。
- 新增本地更新调试日志 `update-debug.log`，更新检查、manifest 拉取、下载与校验失败会落盘，便于问题机回传排查。
- 新增 stable / beta 双更新通道，内测版本使用独立 `latest_test.json` 与 `_test.md` 说明文件。
- 本地授权主机现在会在“同机器指纹 + 同授权约束 + 同通道”时直接复用旧授权码，并尽量回填历史设备名和设备编号。
- 客户端应用本地主机签发的内测授权码后，会自动生成 `update-channel.override.json`，重启后直接进入内测更新通道。
- 主界面、关于页、更新弹窗已补充内测用户 / 测试用户标识与测试标识展示。
- 修复本地授权主机标题栏跟随内容一起滚动的问题，窗口控制条现在固定在顶部。
