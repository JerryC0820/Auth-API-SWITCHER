# 1.0.0

- OAuth 登录链路补强：工作区 OAuth 现在会优先走更接近 Codex 原生的登录流程，源码启动版也补上了浏览器辅助拉起，新登录产生的 `auth.json` 导入、识别和去重链路一起修正。
- 云备份能力补全：新增顶部云备份入口、单空间手动备份、全部空间一键同步，以及卡片级云端状态展示。
- 云备份安全收紧：现在必须先启用二次密码，才能继续使用云备份；老用户如果之前已经存在云备份记录但还没设置二次密码，升级后会先看到提醒。
- 空间卡片安全与导入规则加强：本地导入、同设备旧安全锁升级和异常卡片拦截链路补强，减少复制卡片后直接进入可用状态的风险。
- 卡片管理体验提升：支持批量备注编辑，顶部状态条在滚动时保持可见，新增空间的保存与展示一致性问题也已修复。
- 启动与切换动效更新：加入启动翻牌入场、切换时的全屏磨砂态，以及停用卡片的背面交互细节优化。
- 用户中心、反馈中心与悬浮组件进一步打磨：未读提醒、会话连续性、提示音、透明度与位置设置等体验细节都做了整理。
