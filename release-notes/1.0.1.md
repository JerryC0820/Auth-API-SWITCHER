# 1.0.1

- 新空间发现与当前空间保护修复：重新登录不同账号时，新增空间不再静默顶掉当前空间，发现到新 `auth.json` 后会先走确认导入流程，原卡片会保留。
- 云端备份与同步提醒优化：云端候选改为启动后最多自动提醒一次，其余通过同步面板手动查看；同时加入与本地卡片的去重比对，内容相同的不再重复提示导入。
- 云端导入体验修正：云端备份提醒弹窗改为更统一的宽版样式，导入完成后会立即刷新状态，减少“刚接入又像没加上”的错觉。
- 卡片批量操作补强：新增批量删除卡片，开发者模式新增“已识别停用卡片”的一键启用与恢复停用，减少逐个处理的重复操作。
- 其他稳定性修复：补了一批空间发现、切换后状态刷新和同步细节相关的问题，降低跨设备登录和导入时的误覆盖与重复提示。
