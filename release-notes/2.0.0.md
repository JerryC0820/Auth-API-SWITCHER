# 2.0.0

- 更新链路修复：安装版当前版本统一显示为 `2.0.0`，Windows 在线更新会自动退出旧进程再启动安装器，避免旧进程残留导致版本看起来仍是旧版。
- 更新界面收口：右上角更新按钮补上提醒红点，主界面顶部不再显示下载进度条，更新弹窗移除下载 URL 杂项文案。
- 认证入口重做：扫码页改成独立纯页面式布局，标题移出二维码区域；手动登录/注册/忘记密码重新纵向居中，并补回“记住密码”勾选项与统一返回按钮。
- 用户中心与弹窗统一：启动阶段右侧用户中心改回正式样式节奏，批量删除确认弹窗恢复项目统一风格，并新增“全部删除”确认入口。
- 云端覆盖与删除提速：批量删除和云端覆盖本地改走真正的批量路径；云端全量覆盖本地时会先清空本地卡片，再导入云端，不再逐张慢删。
- 统计与扫码绑定修复：统计中心与空间池共用停用/紧张口径，扫码绑定支持更多上游身份字段结构，缺少身份字段时会返回更明确的定位信息。
