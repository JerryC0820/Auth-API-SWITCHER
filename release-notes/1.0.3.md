# 1.0.3

- 在线更新安装修复：Windows 更新在下载并校验完成后，会直接强制退出当前版本并自动拉起安装器，不再停留在“已下载但还要再点一次安装”的状态。
- 安装器接力兜底加强：如果应用在退出阶段卡住，更新接力脚本会主动强制结束旧进程后再启动安装器，减少更新完成却没有真正覆盖安装的问题。
- 授权遮挡修复：新电脑首次安装且尚未登录时，不再让“发现新空间”等后层弹窗顶在授权验证界面后面，避免登录输入框无法点击和输入。
- 接卡链路修复：接收他人传送的空间卡片时，取消额外的二次密码校验分支，确认接收后会直接进入导入流程。
- 云同步加载卡片对齐：云端下载回主界面后的占位卡，改成与日常双击切换相同的翻面卡样式，统一显示“云端同步加载中”。
- 接卡加载卡片对齐：接收他人卡片后也回到主卡片池，用同样的翻面切换卡样式显示“空间卡片接收中”，逐张成功后再翻成真实卡片。
- 云端备注落地修复：云同步下载到本地时，继续按云端当前可见备注写回卡片，减少公司端落地后又回退成错误备注的情况。
